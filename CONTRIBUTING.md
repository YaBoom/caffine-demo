## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming and inclusive environment for everyone.

## How to Contribute

### Reporting Bugs

- Ensure the bug was not already reported by searching the existing issues
- Use a clear and descriptive title
- Include detailed steps to reproduce the issue
- Specify your environment and version information

### Suggesting Features

- Explain the feature clearly and why it would be valuable
- Consider the impact on existing functionality
- Be open to discussion about implementation approaches

### Pull Requests

1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Add tests if applicable
4. Update documentation as needed
5. Ensure your code follows the project's style guidelines
6. Submit a pull request with a clear description
